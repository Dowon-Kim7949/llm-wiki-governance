## 요약 · Summary

- **읽기 전용 retrieval(Gate 24).** 거버넌스 리포트가 아니라 문서 **본문**을 반환하는 4개 명령을 추가한다 — "에이전트가 매번 코드를 다시 읽는 대신 위키를 query한다"는, 출시 때 철회했던 스토리의 실체. 부가적·opt-in이라 `llm-wiki` 명령 표면·`--format json`·프로그래매틱 API·frontmatter 계약·zero-dep 불변. · Read-only retrieval (Gate 24). Adds four commands that return document **content**, not governance reports — the "the agent queries the wiki instead of re-deriving from code" surface that was walked back at launch. Additive/opt-in; the command surface, `--format json`, the programmatic API, the frontmatter contract, and zero-dep are unchanged.

## 추가 · Added

- **`list-docs`** — status/visibility/doc_type 필터로 문서 메타데이터(path/title/status/doc_type/visibility/last_updated/tags) 열거. 본문 미반환. · Enumerate document metadata with status/visibility/doc_type filters; no bodies.
- **`search-docs <query>`** — 제목·본문·frontmatter에 대한 **zero-dep 키워드/부분문자열** 검색(semantic 아님). 모든 term이 있어야 매치(AND), 점수순 랭크 + 스니펫. `--limit` 기본 20. · Deterministic keyword/substring search (NOT semantic); AND-semantics, ranked, with snippets.
- **`get-doc <path>`** — 문서의 frontmatter + 본문 반환. `<path>`는 repo-relative·wiki-relative·bare name 허용. · Return a document's frontmatter and body by flexible path.
- **`get-related <path>`** — 해소된 그래프 이웃(outbound/inbound: wiki 링크[이중 대괄호]·related·markdown 링크) 반환. · Return a document's resolved graph neighbors.
- **3표면** — 프로그래매틱 API(동결 `commands` 맵, kebab 키)·CLI·MCP(툴 이름 snake_case `list_docs`/`search_docs`/`get_doc`/`get_related`, 다른 MCP 툴과 동일 읽기 전용). · Shipped on the programmatic API, CLI, and MCP.

## 안전 · Safety

- **읽기 전용.** 아무것도 쓰거나 편집·강등하지 않는다. · Read-only; nothing is written, edited, or downgraded.
- **visibility + 민감정보 존중.** restricted/민감 문서(visibility `restricted`·`contains_sensitive_info`·민감정보 스캔 매치)는 `--include-sensitive` 없으면 `list-docs`/`search-docs`에서 **제외**되고, 반환하는 모든 본문/스니펫은 민감 라인을 **redact**한다(raw 값 미반환; `get-doc`은 문서를 반환하되 해당 라인 redact). · Restricted/sensitive docs are excluded from list/search unless `--include-sensitive`; returned bodies/snippets redact sensitive-looking lines (raw values never returned).
- **zero-dependency.** 키워드/부분문자열 + 기존 위키 그래프만 — 임베딩·인덱스·네트워크 없음. · Keyword/substring plus the existing wiki graph only — no embeddings, index, or network.
